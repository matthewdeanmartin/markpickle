# Prior Art and Links

## Similar Libraries

- [markdown-to-json](https://github.com/njvack/markdown-to-json) - Markdown headers to nested JSON/dict (deserialization only)
- [json2md](https://github.com/IonicaBizau/json2md) - DOM-style dict to Markdown (Node.js)
- [tomark](https://pypi.org/project/tomark/) - Dict to Markdown table (unmaintained)
- [pytablewriter](https://pytablewriter.readthedocs.io/) - Dict to Markdown table (and many other formats)
- [keepachangelog](https://pypi.org/project/keepachangelog/) - Single-schema Markdown to dict
