# Examples

## scalar

```python
6
```

```markdown
6
```

## list of scalars

```python
[1, 2, 3]
```

```markdown
- 1
- 2
- 3

```

## list of dictionaries

```python
[{'animal': 'cat', 'name': 'Frisky'}, {'animal': 'dog', 'name': 'Fido'}]
```

```markdown
| animal | name   |
| ------ | ------ |
| cat    | Frisky |
| dog    | Fido   |

```

## dictionaries of strings

```python
{'animal': 'cat', 'name': 'Frisky'}
```

```markdown
# animal
cat
# name
Frisky

```

## dictionary of lists

```python
{'ages': [24, 59, 45], 'countires': ['US', 'Canada', 'Iceland']}
```

```markdown
- ages
 - 24
 - 59
 - 45
- countires
 - US
 - Canada
 - Iceland

```

## dictionary of dictionaries

```python
{'Best Cat': {'animal': 'cat', 'name': 'Frisky'}, 'Best Dog': {'animal': 'dog', 'name': 'Fido'}}
```

```markdown
# Best Cat

 | animal | name   |
 | ------ | ------ |
 | cat    | Frisky |

# Best Dog

 | animal | name |
 | ------ | ---- |
 | dog    | Fido |


```
